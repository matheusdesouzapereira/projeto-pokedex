# projeto-pokedex

Sobre:
- Esse exercício fiz para colocar em prática meus conhecimentos em HTML, CSS3 e Javascript. 
- Nesse projeto eu usei a API do pokeapi.co
- No projeto existem 4 pokemons linstados por padrão e o usuário pode digitar a quantidade que ele quer ver de pokemons, sendo que no momento em que o usuário digita já é possível ver o resultado acontecendo em tempo real.

O que usei:
- HTML
- CSS
- Vanilla Javascript

Não usei nenhum framework.
